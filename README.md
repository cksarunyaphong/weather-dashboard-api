# Weather Dashboard

A weather dashboard built using HTML, CSS, JavaScript, and WeatherAPI.

## Features

- Search any city
- Select popular cities from a dropdown menu
- Display current temperature
- Display weather condition
- Display weather icon
- Show last updated time
- Dynamic background color based on weather

## API Used

WeatherAPI

https://www.weatherapi.com/

## Technologies

- HTML5
- CSS3
- JavaScript
- REST API
